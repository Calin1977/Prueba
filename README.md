## Lenguaje Markdown
El lenguaje Markdown es un lenguaje de texto plano que utiliza una sintaxis para mostrar texto con atributos HTML.

### Por ejemplo:
- [ ] Una casilla vacía
- [X] Una casilla marcada
- Viñetas y listas
  - Elemento 1
    - Elemento 1.1
      - Elemento 1.1.1
  - Elemento 2

Lista de definiciones
<dl>
  <dt>Primer termino</dt>
  <dd>Primera definición</dd>
</dl>
